# 注意力规则

## 收录规则

收录一条资讯前，先问：

1. 它是否改变某个行业、公司、产品或用户行为的约束？
2. 它是否是一个更大趋势的新证据？
3. 它是否能帮助我做判断，而不只是知道一件热闹事？
4. 它是否来自足够可信的来源？

## 降权规则

- 标题党降权。
- 缺少原始来源降权。
- 只复述发布会稿件降权。
- 与长期关注域无关的热点降权。
- 单一社媒爆料未被交叉验证时降权。

## 直接过滤

- 明显营销软文。
- 重复转载。
- 没有事实增量的观点文。
- 只提供情绪、没有信息的争吵。

## 播报规则

- 每条都必须说明“为什么重要”。
- 不要用新闻腔。
- 不要把公司 PR 当事实。
- 不确定就说不确定。
- 对同一事件，要合并多来源，避免重复播报。
